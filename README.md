// Za pierwszym razem zaleca się odświeżenie pamięci cache.
// Aby odświeżyć pamięć podręczną (cache) strony, użyj skrótu klawiaturowego Ctrl + F5 lub Ctrl + R. 
// Kombinacja Ctrl + F5 wymusza ponowne pobranie strony z serwera, pomijając lokalnie zapisaną kopię. 
// Sam skrót F5 odświeża stronę, ale może wykorzystać dane z cache'u, natomiast Ctrl + R odświeża aktualne okno, 
// a Ctrl + Shift + R w niektórych przeglądarkach również wymusza twarde odświeżenie. 
// Skrót Ctrl + Alt + R nie jest standardowym skrótem do odświeżania cache w przeglądarkach.
